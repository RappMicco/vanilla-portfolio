JPF Ordering System


JPF (Joegen Piggery Farm Ordering System) is a web-based application developed to automate and manage affiliate order transactions for a piggery farm. The system replaces the former manual process where orders were handwritten in notebooks and calculations were performed manually, resulting in inefficiencies and potential errors. JPF provides automated order computation, PDF generation for downloadable order summaries, and centralized data management. It implements user authentication with secure login and user registration functionalities. Additionally, the system supports full CRUD (Create, Read, Update, Delete) operations to ensure accurate and maintainable records.

Wi-Fi Integrated Smart home using ESP32 and ESP-RainMaker


WISE (Wi-Fi Integrated Smart Home using ESP32 and ESP-Rainmaker) is a smart home system designed to help homeowners manage electrical devices efficiently, reduce electricity costs, and improve home safety—especially for seniors and individuals who are frequently away from home. The system enables real-time monitoring of electrical device usage and supports remote control from anywhere via internet connectivity. WISE integrates voice control through Google Assistant and Amazon Alexa, allowing users to turn devices on or off using voice commands. It features a user-friendly interface suitable for all age groups and implements secure remote access to ensure data privacy and prevent unauthorized device control.
